## أرشيف 1: التحقق النهائي من Firestore (مركونة بطلب المالك)
الكود كامل ومدفوع (المصادقة + اختبار الاتصال). المتبقي على المالك عند العودة:
1. Firestore Database → Rules → لصق قواعد `request.auth != null` → Publish
2. Authentication → Sign-in method → التأكد من تفعيل Google
3. Authentication → Settings → Authorized domains → إضافة slomoo-94.vercel.app
4. فتح التطبيق → تسجيل الدخول → تشغيل «اختبار الاتصال» في المستودع → توقع ✓ الأخضر
