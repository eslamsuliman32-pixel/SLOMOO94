# مقام (MAQAM)

مساعد رقمي ومدرب شخصي متكامل لصناعة الراب العربي — للاستخدام الشخصي.
**المبدأ الحاكم: الآلة تكشف وتعرض وتقيس — الإنسان يقرر ويكتب ويؤدي.**

## الأركان الأربعة
التدريب · الاستوديو الإبداعي · التحليل الموسيقي · المستودع المعرفي — تربطها شخصية مدرب واحدة وقاعدة بيانات واحدة.

## هيكل المشروع
- `frontend/` — تطبيق React (الواجهة، Web Audio، الهوية البصرية Techno-Arabic Brutalism)
- `analysis-lab/` — نوتبوك معمل التحليل (Kaggle/Colab: Demucs، madmom، Basic-Pitch، Essentia)
- `scripts/` — سكريبتات النقل والصيانة (تصدير/استيراد البيانات)
- `docs/` — الوثائق الحاكمة (تُقرأ قبل أي كود)
- `docs/knowledge-base/` — المراجع المعرفية (أسرار اللغة، دلالة الألفاظ، موسيقى الشعر)

## الوثائق الحاكمة
1. `docs/PLAN.md` — الخطة التنفيذية (66 خطوة مرقمة)
2. `docs/SYSTEM_MAP.md` — مصير مكونات النظام القديم وبروتوكول الرجوع للأرشيف
3. `docs/RHYME_PROSODY_SPEC.md` — المنهجية القافوية والوزنية (الدمج المعرفي: عروض الخليل بلغة ●/▬)
4. `docs/BEAT_PROFILE_SPEC.md` — عقد بيانات التحليل الصوتي
5. `docs/DECISIONS.md` — سجل القرارات المحسومة
6. `docs/MODULE_CONTRACT.md` — عقد الوحدات الموحد
