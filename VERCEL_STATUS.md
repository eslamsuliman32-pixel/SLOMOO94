## 🎯 خطوات النشر على Vercel (ملخص سريع)

### الطريقة الأولى: النشر الفوري (الموصى به)

```bash
# 1. تسجيل الدخول إلى Vercel
npm i -g vercel

# 2. النشر المباشر من المشروع
vercel

# 3. متابعة التعليمات على الشاشة
# اختر:
# - Project name: SLOMOO94-EM-Semantics
# - Root directory: ./frontend
# - Framework: Vite
```

### الطريقة الثانية: عبر لوحة Vercel

1. اذهب إلى: https://vercel.com/dashboard
2. اضغط: "Add New" → "Project"
3. اختر: "Import Git Repository"
4. ابحث عن: `eslamsuliman32-pixel/SLOMOO94`
5. في الإعدادات:
   - Root Directory: `./frontend`
   - Build Command: `npm run build`
   - Output: `dist`
6. اضغط: "Deploy" 🚀
